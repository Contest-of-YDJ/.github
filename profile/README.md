"**Global Connect**"!
외국인 근로자를 위한 산재 도움 예방 웹 서비스

공공데이터 API를 사용하여 산재 예방 안내 및 산재 지원 병원 위치와 안전 사업장 정보를 공유하여 도움을 주고자 하는 서비스입니다.

## **🔧 Tech stack**

### Back-end

- Java 17
- Gradle 7.6.1
- Spring Boot 3.0.5
- JWT
- Spring Data JPA
- MySQL 8
- Lombok
- Swagger 3.x
- 고용노동부 API
- Apache PDFBox 2.0.26

### Front-end

- React
- JavaScript
- Axios
- eslint
- Kakao Map API
